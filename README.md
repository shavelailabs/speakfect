# Speakfect — public legal pages

  This repository hosts the public-facing legal documents for the
  [Speakfect](https://speakfect.ai) mobile app.

  GitHub Pages serves the site from the `docs/` folder of `main`:

  - Privacy Policy — [/privacy/](https://shavelailabs.github.io/speakfect/privacy/)
  - Terms of Service — [/terms/](https://shavelailabs.github.io/speakfect/terms/)

  The full Speakfect product source code lives in a separate Replit workspace
  and is not published here. Only the documents below are public:

  - `docs/privacy-policy.md` — current Privacy Policy
  - `docs/terms-of-service.md` — current Terms of Service
  - `docs/index.md` — landing page
  - `docs/_config.yml` — Jekyll config (theme + exclusions)

  For questions: **support@speakfect.ai** / **privacy@speakfect.ai**
